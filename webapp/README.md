# Réalisons Webapp

## Prérequis
- Node.js 18+
- npm 9+

## Installation
```bash
cd webapp
npm install
```

## Développement
```bash
npm run dev
```
L'application est servie sur `http://localhost:3000` avec Tailwind CSS.

## Build de production
```bash
npm run build
npm run start
```

## Tests de style
```bash
npm test
```
Exécute les tests Jest/Testing Library qui vérifient la présence des classes Tailwind essentielles.
